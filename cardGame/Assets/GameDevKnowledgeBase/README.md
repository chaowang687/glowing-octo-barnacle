# 游戏开发知识库

## 概述
本知识库包含了项目中各个功能模块的详细文档，旨在帮助开发人员快速了解和应用这些功能模块到其他项目中。

## 模块列表

### 核心系统
- [背包系统](背包系统.md) - 完整的物品管理系统，支持异形物品、360度旋转、自动寻找空位、星星连接高亮等功能
- [卡牌系统](卡牌系统.md) - 完整的回合制卡牌战斗系统，支持能量管理、卡牌堆栈操作、多目标选择等功能
- [战斗系统](战斗系统.md) - 完整的回合制战斗系统，支持回合管理、卡牌打出、敌人AI协调等功能
- [地图系统](地图系统.md) - 完整的地图管理系统，支持地图生成、节点管理、场景跳转等功能

### 辅助系统
- [本地化系统](本地化系统.md) - 完整的本地化管理系统，支持多语言切换、CSV文件管理等功能
- [宝箱搜刮系统](宝箱搜刮系统.md) - 完整的宝箱搜刮系统，支持战利品生成、动画效果等功能
- [角色选择系统](角色选择系统.md) - 完整的角色选择系统，支持角色预览、选择、开始游戏等功能
- [挖掘系统](挖掘系统.md) - 完整的挖掘系统，支持网格挖掘、化石收集、物品飞行动画等功能

## 使用指南

### 1. 如何使用知识库
1. 浏览模块列表，找到你需要的功能模块
2. 点击模块链接，查看详细文档
3. 按照文档中的使用示例和API说明，将功能模块应用到你的项目中

### 2. 如何扩展知识库
1. 在`Assets/GameDevKnowledgeBase/`目录下创建新的markdown文件
2. 按照现有文档的结构编写新模块的文档
3. 在本README.md文件中添加新模块的链接

### 3. 文档结构说明
每个模块文档包含以下部分：
- **模块概述**：简要介绍模块的功能和用途
- **核心功能**：列出模块的主要功能
- **核心文件和类**：列出模块的核心文件和类，包括位置和功能
- **关键API和方法**：列出模块的关键API和方法
- **使用示例**：提供模块的使用示例代码
- **配置和依赖**：列出模块的配置和依赖
- **常见问题和解决方案**：提供常见问题的解决方案
- **扩展建议**：提供模块的扩展建议

## 注意事项

1. 本知识库基于当前项目的实现，可能需要根据你的项目需求进行适当的调整
2. 部分功能模块可能依赖于项目中的其他组件，请确保在使用时正确引用
3. 如有任何问题或建议，请随时更新知识库文档
